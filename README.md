# Kenken Solver

This simple web app generates random Kenken boards and solves them using various techniques.

## How to start the app?

To run the app you need to first have python installed, then install the requirements as follows:

    pip install -r requirements.txt

Then you should start the app as follows:
	
    python main.py

You should then be able to see the app on localhost:5000

To run evaluation tests run the following command:
	
	python main_evaluate.py

Evaluation may take a long time.
